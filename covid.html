Covid 19 Data Exploration

Tableau | SQL Server


Making The Dashboard

COVID-19 was the most impactful disease of the 21st century. As a result, I developed a deep curiosity for the data that the CDC had accrued on the disease that shut down the world. Using Tableau, I was able to visualize this data. Specifically, we look at COVID 19's impact for the year of 2022.

Key Components:
  • A custom table listing the total cases and deaths as well as the death percentage.
  • A heat map showing which countries have the highest infection rate.
  • A bar chart comparing death count by continent.
  • A line graph showing the percent of population infected up until July 2022 and a prediction for the rest of the year.

Preparing for Visualizing

Prior to visualization, I cleaned and prepared the dataset using SQL server. Keeping my code organized and easy to read is always a top priority for me. Using comments and empty space, I'm able to present my code to potential team members with ease and maximize efficiency.

Key Components:
  • Common Table Expressions (CTE's)
  • Temp Tables
  • Custom Views
